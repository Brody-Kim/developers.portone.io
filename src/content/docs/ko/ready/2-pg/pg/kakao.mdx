## 일반**결제**

<Tabs>
<Tab title="테스트 결제">
**테스트 환경 구성방법**

1. [포트원 관리자 콘솔](https://admin.portone.io/)→ 결제연동 → 테스트 연동 관리 →카카오페이 -> 카카오페이->**추가** 선택

## 정기결제

<Tabs>
<Tab title="테스트 결제">
**테스트 환경 구성방법**

1. [포트원 관리자 콘솔](https://admin.portone.io/)→ 결제연동 → 테스트 연동 관리 →**카카오페이** -> **카카오페이**-><mark style="color:red;">**추가**</mark> 선택
