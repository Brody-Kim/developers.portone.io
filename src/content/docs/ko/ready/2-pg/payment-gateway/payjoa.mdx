## 인증**결제**

<Tabs>
<Tab title="테스트 결제">
**테스트 환경 구성방법**

1. [**포트원 관리자콘솔**](https://admin.portone.io) 로그인 -> **결제 연동** -> **테스트 연동 관리** -> **페이조아** -> <mark style="color:red;">**페이조아**</mark> 선택->**추가**
2. 자동으로 기재되는 상점 ID/ 결제연동키/ MID 확인 후 **저장** 선택

<Figure src="/gitbook-assets/ko/image (77) (1).png" caption="포트원 관리자콘솔 로그인 -> 결제 연동 -> 테스트 연동 관리 -> 페이조아 -> 페이조아 선택->추가" />

<Figure src="/gitbook-assets/ko/image (8).png" caption="자동으로 기재되는 상점 ID/ 결제연동키/ MID 확인 후 저장 선택" />

</Tab>

<Tab title="실 결제">
**실 환경 구성 방법**

1. [**포트원 관리자콘솔**](https://admin.portone.io) 로그인 -> **결제 연동** -> **실 연동 관리** 선택
2. **페이조아** -> <mark style="color:red;">**페이조아**</mark> 선택 -> **추가**
3. 다우 PG사 계약 후 발급받은 상점 정보를 기재 합니다.

<Figure src="/gitbook-assets/ko/image (46).png" caption="포트원 관리자콘솔 로그인 -> 결제 연동 -> 실 연동 관리 선택" />

<Figure src="/gitbook-assets/ko/image (91).png" caption="페이조아 -> 페이조아 선택 -> 추가" />

<Figure src="/gitbook-assets/ko/image (9) (1).png" caption="다우 PG사 계약 후 발급받은 상점 정보를 기재" />

<Hint style="success">
**확인사항**

* 상점아이디 발급 시 **판매상품 유형**(실물 / 디지털)을 확인하시고 발급요청 해야 합니다.
* **결제연동 키**는 결제 취소 시 필요한 키 값을 의미합니다.

</Hint>



**\[페이조아 관리자 콘솔 설정 사항]**

각 결제수단별 **정상적인 결제 결과를 수신**하기 위한 아래 설정이 반드시 필요합니다.

1. [페이조아 관리자페이지](https://agent.kiwoompay.co.kr/) 로그인
2. 상단 **고객지원** > **연동정보설정** 메뉴 클릭

![](</gitbook-assets/ko/image (44).png>)

3\. 연동 정보 설정 화면에서 하단 **CPID**를 선택 후 **조회하기** 버튼 선택

<Figure src="/gitbook-assets/ko/image (279).png" caption="연동정보 설정" />

4\. 계약된 결제수단 별로 하단의 **결과통지 URL 설정**

> [https://service.iamport.kr/daou\_payments/result](https://service.iamport.kr/daou\_payments/result)

![](</gitbook-assets/ko/image (70).png>)

</Tab>
</Tabs>

\- Secret : 7f59ab6239259b191e2806111734c17c4650301ae0f241d2939c747176b4f6aa

* **카드번호 + 유효기간**

\- CPID : <mark style="color:red;">**CTS16628**</mark>

\- Secret : 9e810e78e4d8ac765f8f04f1f3132da0c4891a2c04f6ee5a142bc333070b68a9

<Hint style="success">
**확인사항**

* 페이조아의 정기결제는 REST API 방식만 지원합니다.
* 페이조아 테스트 모드의 경우 실제 출금 되지만 매일 23:00\~23:50분 사이 자동 취소됩니다.
* **키인결제**(일회성결제)와 **정기결제용** 테스트 상점아이디가 상이하니 유의하여 연동해주세요!
* 상점아이디 발급 시 판매상품 유형(실물 / 디지털)에 맞게 발급 받아 주세요.

</Hint>
