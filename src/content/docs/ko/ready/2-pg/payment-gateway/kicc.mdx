## 인증**결제**

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) → **결제 연동** -> **테스트 연동 관리** → **KICC** → <mark style="color:red;">**KICC**</mark> 선택 → 추가
2. 자동으로 기재된 MID 확인후 **저장** 선택

</Tab>

<Tab title="실 결제">
#### **실** 환경 구성방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) → **결제 연동** → **실 연동관리** 선택
2. **KICC** → <mark style="color:red;">**KICC**</mark> 선택 → **추가**
